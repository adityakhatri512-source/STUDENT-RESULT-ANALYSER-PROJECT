# STUDENT-RESULT-ANALYSER-PROJECT

The Student Result Analyzer is a C++ based mini project developed to simplify the process of managing and analyzing student academic results. The system allows users to store student details such as name, roll number, and subject marks, and automatically calculates total marks, percentage, grade, and pass/fail status. It helps reduce manual calculation errors and improves the efficiency of result management in educational institutions.

This project is designed using Object-Oriented Programming concepts in C++ including classes, functions, loops, arrays, and file handling. The application also stores result data in a text file for record maintenance. The main objective of the project is to provide a simple, accurate, and organized solution for student performance analysis while giving practical exposure to real-world software development concepts.

Key Features:

Student record management
Automatic percentage and grade calculation
Pass/Fail evaluation
Result display system
File handling for data storage
Simple and user-friendly console interface
